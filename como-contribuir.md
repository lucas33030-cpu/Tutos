# ✏️ Como Contribuir com os Tutoriais

> Qualquer membro da equipe pode melhorar esta base de conhecimento. É mais fácil do que parece!

---

## 📝 Editando um tutorial existente

1. Acesse o repositório no GitHub
2. Navegue até o arquivo que deseja editar (pasta `tutoriais/`)
3. Clique no ícone de **lápis** ✏️ no canto superior direito
4. Faça suas alterações no texto (formato Markdown)
5. No final da página, clique em **"Commit changes"**
6. Pronto! As alterações aparecem no site em segundos

---

## ➕ Criando um novo tutorial

1. No repositório, navegue até a pasta `tutoriais/`
2. Clique em **"Add file" → "Create new file"**
3. Nomeie o arquivo: `nome-do-tutorial.md`
4. Escreva o conteúdo usando Markdown
5. Faça o commit
6. Adicione o link no arquivo `_sidebar.md` para aparecer no menu

---

## 📖 Guia rápido de Markdown

```markdown
# Título principal
## Subtítulo
### Seção menor

**negrito**   _itálico_

- item de lista
- outro item

1. lista numerada
2. segundo item

[texto do link](url)

> citação ou destaque

| Coluna 1 | Coluna 2 |
|---|---|
| dado | dado |
```
